## Long Description

Fluxbreak is a wave-based roguelike auto-battler set in deep space. Your ship fights continuously while you steer the run through positioning, upgrades, and tactical docking choices.

Key features:
- Auto-combat focused on weapon and upgrade synergy
- Escalating wave pressure with adaptive threat pacing
- Multiple ship and pilot stat profiles
- Fast session loop with persistent high-score progression

The challenge is not only firepower. Surviving means deciding when to commit, when to kite, and how to build around the tools the run gives you.
