FLUXBREAK - PLAYER README

Version: 0.1.5
Support: p4ndepravity@gmail.com

INSTALLATION
1) Run the installer and follow prompts.
2) Launch Fluxbreak from the Start Menu shortcut.

SYSTEM NOTES
- This build targets 64-bit Windows.
- Save/config data is written to your user profile.

DEFAULT CONTROLS
- Thrust: W
- Reverse: S
- Roll Left: Q
- Roll Right: E
- Boost: Shift
- Pause: Esc
- Hull Repair: R
